Etiqueta Mark

El Elemento Mark representa un texto marcado o resaltado como referencia o anotación , debido a su relevancia o importancia en un contexto particular.

Notas de uso

  • En una cita () o en otro bloque "blockquote", generalmente indica texto que es de especial interés pero que no está destacado en el material de origen, o material que necesita especial escrutinio, aunque el autor original no lo considerase importante.

  • Por otro lado, indica una parte del contenido del documento que probablemente sea relevante para el usuario. Por ejemplo, se puede utilizar en una página que muestra los resultados de búsqueda para resaltar cada uno de estos por palabra.

  • No uses el elemento para resaltado de sintaxis; usa el elemento para este fin.
